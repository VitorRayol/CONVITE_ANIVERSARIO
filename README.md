# 🌟 Heitor Star Wars Baby - Convite de Aniversário

Site de convite digital para o primeiro aniversário do Heitor com tema Star Wars Baby.

## 🚀 Tecnologias

- **Next.js 14** - Framework React
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Estilização
- **Framer Motion** - Animações
- **Supabase** - Banco de dados
- **Lucide Icons** - Ícones

## 📋 Pré-requisitos

- Node.js 18+
- Conta no Supabase
- (Opcional) Conta no WhatsApp Business API
